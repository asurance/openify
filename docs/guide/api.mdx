# API

## openify

- 入参
  - comp：对应的弹窗组件
  - config(可选)：打开时的配置，参考[OpenifyConfig](#openifyconfig)
- 返回值
  - 返回一个 Promise，对应内容为弹窗组件`onClose`带的第一个参数
- 属性
  - defaultRenderHook：默认的渲染钩子，参考[OpenifyRenderHook](#openifyrenderhook)

## OpenifyConfig

- 属性
  - transformProps：Props 转换钩子，针对某些不符合[OpenableProps](#openableprops)用于将`openify`生成的属性转换成组件需要的属性

## OpenifyRenderHook

- 入参
  - node：`openify`生成的组件节点，用于给全局 context 或自定义 ErrorBoundary 包裹
  - onError：组件内部抛错的回调，一般通过 ErrorBoundary 触发，用于出错后清理内存
- 返回值
  - 返回一个 ReactElement，对应实际渲染的内容

## OpenableProps

- 属性
  - visible：组件是否可见
  - onClose：组件关闭回调，可带一个参数
  - afterClose：组件关闭结束回调
